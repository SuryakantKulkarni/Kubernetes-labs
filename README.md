## 📌 About This Repository

This repository contains my hands-on Kubernetes lab solutions and troubleshooting notes from KodeKloud practice labs.

The main goal of this repository is to document:
- ✅ Kubernetes concepts through practical labs
- ✅ Real command-line workflows
- ✅ YAML configurations and manifests
- ✅ Debugging and troubleshooting approaches
- ✅ Key learnings from every task

Every task includes:
- Step-by-step implementation
- Command explanations
- YAML manifests
- Troubleshooting notes
- Key Kubernetes concepts learned

This repository is focused on practical Kubernetes administration and real-world troubleshooting instead of only theory.
